Каскадные таблицы стилей: CSS

Каскадные таблицы стилей описывают внешний вид документа, написанного с помощью языка разметки (HTML).

CSS-свойства

С помощью CSS-свойств задают внешний вид страницы: размер шрифта, его начертание, размеры блоков, способы их отображения, отступы, цвет текста, создают анимацию и многое другое.
список свойств CSS
CSS-свойства можно задавать несколькими способами:

  • в атрибуте style HTML-тега
  • с помощью правил внутри тега style
  • с помощью правил в подключенном файле с расширением *.css
CSS-свойства задаются таким образом: свойство: значение;
Правила хорошего тона предписывают нам НЕ задавать CSS-свойства с помощью атрибута style без необходимости.

CSS-правила и селекторы

Правила CSS объединяют в себе набор свойств и выглядят так:

          селектор, селектор... {
            свойство: значение;
            свойство: значение;
            ...
          }

Селекторы бывают разных видов и определяют, для каких элементов определить описанные свойства.
список селекторов CSS
Основные виды селекторов:

  • Универсальный селектор - устанавливает стиль для всех элементов страницы. 
                  * {
                ...
              }
  • Селектор тегов - устанавливает стиль для всех тегов с заданным именем.
    В примере для всех тегов h2 задан цвет текста. 
                  h2 {
                color: #5E8DB2;
              }
  • Селектор классов - устанавливает стиль для всех тегов с заданным классом.
    Класс тега задается с помощью атрибута class. 
                  .code {
                font-family: 'Courier New', Courier, 'Lucida Sans Typewriter', 'Lucida Typewriter', monospace;
                font-weight: bold;
                color: #FF5722;
                font-size: 1.2em;
              }


              span class="code"class/span
  • Селектор идентификаторов - устанавливает стиль для всех тегов с заданным идентификатором.
    Идентификатор тега задается с помощью атрибута id. 
                  #tag-types {
                text-decoration: underline;
              }
  • Селектор атрибутов - устанавливает стиль для всех тегов с определенным значением указанного атрибута.
    В примере для всех тегов a с атрибутом target="_blank" устанавливается цвет текста. 
                  a[target="_blank"] {
                color: #109684;
              }
  • Селектор потомков - устанавливает стиль для всех элементов, являющимися потомками заданного элемента независимо от вложенности.
    В примере: для всех тегов span, являющихся потомками тега ul с классом list-none установлен жирный шрифт. 
                  ul.list-none span {
                font-weight: bold;
              }
  • Селектор дочерних элементов - устанавливает стиль для всех элементов, являющимися прямыми потомками заданного элемента.
    В примере: для всех тегов li, являющихся прямыми потомками тега ul с классом list-none установлен цвет текста для первой буквы. 
                  ul.list-none > li::first-letter {
                color: #5E8DB2;
              }
  • Селектор псевдоклассов - устанавливает стиль для различных состояний элементов, например, для ссылки при наведении на нее курсора. 
                  a:hover {
                color: #5ADAC9;
              }
  • Селектор псевдоэлементов - устанавливает стиль элементов, отсутствующих в дереве документа. Например, можно определить стиль первой буквы содержимого элемента или дописать контент до или после содержимого элемента. 
                  ul.list-none > li::first-letter {
                color: #5E8DB2;
              }
              .tag::before {
                content: "<";
              }
              .tag::after {
                content: ">";
              }

Каскадирование

Каскад в CSS - это одновременное применение различных стилей к элементам документа. Для того, чтобы при отображении страниц не возникало конфликтов, существуют приоритеты, которыми пользуются браузеры при отображении страниц. Список составлен от меншего приоритета к большему.

  1. Стиль браузера: оформление, по умолчанию применяющееся браузером к страницам.
  2. Стиль автора: стили, прописанные разработчиком.
  3. Стиль пользователя: стили, написанные самим пользователем.
  4. Стиль автора с добавлением !important
  5. Стиль пользователя с добавлением !important
Ключевое слово !important повышает приоритет приоритет свойства, для которого указано. Разработчику чаще всего можно обойтись без примерения !important.

На приоритет также оказывает влияние специфичность селектора, которая вычисляется по следующеу правилу.

Универсальный селектор (*) 0
Селектор тега и псевдоэлемента 1
Селектор класса и псевдокласса 10
Селектор идентификатора 100
Стиль через атрибут style 1000

Для каждого правила вычисляется сумма и устанавливается приоритет (чем больше сумма, тем выше приоритет). Этот абзац является примером использования каскадирования.

Если в документе объявлено несколько правил с одинаковыми селекторами, то применится то, которое описано ниже.

Отступы и границы

С помощью CSS можно задавать отступы для элементов, для этой цели существует несколько CSS-свойств.
  • свойство margin - устанавливает величину отступа от каждого края элемента до внутреней границы его родительского элемента.
  • свойство padding - устанавливает отступ вокруг сожержимого элемента.
  • свойство text-indent устанавливает отступ красной строки.
Значения свойств margin и padding можно задавать несколькими способами:
  1. margin: 20px и padding: 20px задает одинаковые отступы во всех сторон
  2. margin: 10px 20px и padding: 10px 20px задает отступы сверху и снизу (10px) и справа и слева (20px)
  3. margin: 10px 20px 15px и padding: 10px 20px 15px задает отступы сверху (10px), справа и слева (20px) и снизу (15px)
  4. можно задавать отступы для каждого края отдельно с помощью свойств
    margin-left, margin-riht, margin-bottom, margin-top,
    padding-left, padding-riht, padding-bottom, padding-top

Если требуется заключить элемент в рамку, используют свойство border.
Значение этого свойства складывается из трех параметров: ширины границы, ее стиля и цвета. Например, border: 2px solid #5E8DB2; означает, что ширина границы: 2 пикселя, она сплошная и синего (#5E8DB2) цвета.
Все эти параметры можно задать отдельно, используя свойства border-width, border-style, border-color.

Блочные и встроенные элементы

Блочные элементы отображаются на странице в виде прямоугольников. Такие элементы всегда начинаются с новой строки, занимают всю свободную ширину, а их высота определяется их содержимым. К таким элементам относятся div, p, теги заголовков и др.
Особенности блочных элементов:

  • к ним применимы все отступы
  • всегда начинаются с новой строки

Встроенные элементы являются частью других элементов и используются для изменения отображения текста. К таким элементам относятся span, a, b и др. Особенности встроенных элементов:

  • к ним применимы только левые и правые отступы (верхние и нижние будут игнорироваться)
  • игнорируют указание ширины и высоты
  • позволяют обтекание слева и справа

С помощью свойста display можно менять способ отображения элемента. Основные значения, которые может принимать это свойство:

  • block - элемент будет отображаться как блочный. Например:

    Это тегspan, по умолчанию он является встроенным элементом. Если установить свойство display: block, мы увидим, как изменится отображение этого элемента, и каждый новый элемент будет находиться на новой строке.

  • inline - элемент будет отображаться как встроенный. Например:

    Это тег p, по умолчанию он является блочным элементом.

    Если установить свойство display: inline, у него пропадут отсупы

    и каждый такой элемент не будет начинаться с новой строки.

  • inline-block - элемент будет обладать как свойствами блочного, так и встроенного элемента. Это означает, что:
    • такому элементу можно задать высоту и ширину, и к нему применимы все отступы, как к блочному
    • такой элемент позволяет обтекание справа и слева, как встроенный

    Использование такого значения можно посмотерть на этом примере. У элементов div задано свойство display: inline-block и они отображаются друг за другом (обтекание), но с сохранением всех отступов и заданной шириной.

  • none - элемент не будет отображаться на странице, а сама страница будет формироваться так, будто этого элемента нет.

    display: none;

Обтекание элемента

С помощью свойства float можно задать, по какой стороне будет происходить выравнивание элемента и обтекание его остальными элементами.

Элемент может быть не выровнен (значение none) или выровнен по левому или правому краю (значения left или right).

Для примера зададим обтекании картинки текстом. У фотографии с пляжем задано свойство float: left;, поэтому она выравнивается по левому краю, а этот текст обтекает ее с правого края.

У фото с черепахой задано свойство float: right;, поэтому изображение выравнивается по правому краю, а текст обтекает его слева. Для выравнивания текста по правому краю задано свойство text-align: right;.

У фото с лисой задано свойство float: none;, поэтому этот текст отображается под картинкой. Такое значение задано у элементов по умолчанию.

Для отмены обтекания используется свойство clear, Оно может принимать значения:

  • none - обтекание будет происходить в соответствии со свойством float.
  • left - отменяет обтекание с левого края.
  • right - отменяет обтекание с правого края.
  • both - отменяет обтекание одновременно с левого и правго края.

Позиционирование элементов

В CSS можно задавать позиционирование элемента, то есть его положение относительно других элементов. Это можно сделать с помощью свойства position. Это свойство может принимать следующие значения:

  • static: значение по умолчанию, элементы отображаются в том порядке, в котором описаны;
  • absolute: элемент позиционируется с помощью свойств left, right, top и bottom относительно родительского элемента (первого не static);
    Пример использования свойства
  • fixed: элемент позиционируется относительно окна браузера с помощью свойств left, right, top и bottom;
    Пример: кнопка "наверх" в правом нижнем углу окна;
  • relative: элемент позиционируется относительно его положения в документе с помощью свойств left, right, top и bottom.

Визуальные эффекты

С помощью CSS можно создавать множество визуальных эффектов. Рассмотрим некоторые из них.
  • Степень прозрачности элемента задается свойством opacity. Это свойство принимает значения от 0 до 1, где 0 - полностью прозрачный элемент, а 1 - непрозрачный. Примером может служить кнопка "наверх".
  • Эффект перехода между состояниями элемента задается свойством transition. Пример: при наведении на кнопку "наверх", у нее медленно меняется значение свойства opacity.
  • Масштабирование изображений задается свойством object-fit. Это свойство может принимать значения:

    fill (элемент целиком умещается в заданные размеры, пропорции игнорируются)

    contain (элемент целиком умещается в заданные размеры с соблюдением пропорций)

    cover (элемент увеличивается или уменьшается, чтобы целиком заполнить заданную область с сохранением пропорций)

    none (cохраняются исходные пропорции элемента, значения ширины или высоты не влияют на содержимое)

    Положение изображения задается с помощью свойства object-position. Пример использования свойства.
  • Тень к тексту и элементу. Тень к тексту добавляется с помощью свойства text-shadow, тень к элементу - с помощью свойства box-shadow.
    Оба свойства задаются таким образом: <сдвиг по x> <сдвиг по y> <радиус размытия> <цвет>. Пример использования свойства.

Медиа-запросы

Для поддержки различных технических параметров устройств используются медиа-запросы. Такие запросы начинаются с правила @media с указанием условия, при котором будут применяться указанные стили.
Это могут быть:
  • тип носителя, например, projection, screen, print и другие.
    Пример использования:

    Эта надпись выводится только при отбражнии на экранах мониторов.

    Эта надпись выводится только при печати.

    Эта надпись выводится только при отбражнии на проекторе.

  • ширина экрана: для создания адаптивного дизайна. Можно указать предельные значения размера экрана для исползуемых стилей, например:
  • ориентация экрана - orientation (landscape или portrait )
и многие другие.

В условиях могут находиться логические операторы, такие как: and, or, not и only.
Вместо оператора or можно использовать запятую.
Оператор only скрывает правило от старых браузеров.
