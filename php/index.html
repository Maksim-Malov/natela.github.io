Установка PHP+MySQL, phpMyAdmin

Для того, чтобы начать работу с PHP и MySQL на Cloud9 нужно:

  1. Зайти на Cloud9.
  2. Создать workspace, при создании выбрать шаблон: PHP. Открыть workspace.
  3. Открыть консоль (Alt+T) и выполнить последовательно команды:

    mysql-ctl start
    phpmyadmin-ctl install

  4. Перейти по ссылке, вернувшейся в консоли (ссылка вида https://workspacename-username.c9users.io/phpmyadmin).
  5. Авторизоваться, используя логин Cloud9 и пустой пароль. Перед вами должна открыться главная страница phpMyAdmin. С помощью кнопки "Change password" можно установить пароль.
